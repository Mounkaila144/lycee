# Story 6.4: Export de la Structure Académique

## Status: Aproved

## Story

**As an** Admin,
**I want** exporter la structure académique en format imprimable,
**so that** je puisse archiver ou partager la structure avec les autorités académiques.

## Acceptance Criteria

1. Un bouton "Exporter" est disponible sur la vue d'ensemble
2. Options d'export :
   - PDF : Document formaté avec la structure complète
   - Excel : Tableur avec onglets (Classes, Matières, Coefficients, Affectations)
3. L'export inclut : Liste des classes avec PP et effectifs, Tableau des coefficients, Liste des affectations enseignants, Barèmes configurés
4. L'export est filtré par année scolaire
5. Le document généré porte l'en-tête de l'établissement

### Export PDF
```php
public function export(Request $request)
{
    $yearId = $request->academic_year_id ?? AcademicYear::active()->first()->id;
    $format = $request->format ?? 'pdf';

    $data = [
        'academicYear' => AcademicYear::find($yearId),
        'classes' => Classe::where('academic_year_id', $yearId)->with([...])->get(),
        'subjects' => Subject::active()->get(),
        'coefficients' => SubjectClassCoefficient::with([...])->get(),
        'assignments' => TeacherSubjectAssignment::where('academic_year_id', $yearId)->with([...])->get(),
        'settings' => ClassSetting::first(),
    ];

    if ($format === 'pdf') {
        $pdf = Pdf::loadView('documents.structure-export', $data);
        return $pdf->download("structure-{$data['academicYear']->name}.pdf");
    }

    return Excel::download(new StructureExport($data), "structure-{$data['academicYear']->name}.xlsx");
}
```

### Export Excel - Multi Sheets
```php
class StructureExport implements WithMultipleSheets
{
    public function sheets(): array
    {
        return [
            'Classes' => new ClassesSheet($this->data),
            'Matières' => new SubjectsSheet($this->data),
            'Coefficients' => new CoefficientsSheet($this->data),
            'Affectations' => new AssignmentsSheet($this->data),
            'Barèmes' => new SettingsSheet($this->data),
        ];
    }
}
```
