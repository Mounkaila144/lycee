# Story 2.4: Affectation du Professeur Principal

## Status: Aproved

## Story

**As an** Admin,
**I want** affecter un professeur principal à chaque classe,
**so that** chaque classe ait un référent responsable du suivi des élèves.

## Acceptance Criteria

1. Lors de la création/modification d'une classe, je peux sélectionner un professeur principal
2. La liste des enseignants disponibles ne montre que les enseignants qui ne sont pas déjà PP d'une autre classe pour cette année scolaire
3. Si un enseignant est déjà PP d'une autre classe, il n'apparaît pas dans la liste (ou apparaît grisé)
4. Le changement de professeur principal est possible à tout moment de l'année
5. Un récapitulatif affiche les classes sans professeur principal (alerte orange)
6. Le professeur principal est affiché dans le détail de la classe et dans la liste des classes

### Contrainte Unicité PP [Source: docs/prd/module-structure-academique.md#FR20]
Un professeur principal ne peut être affecté qu'à une seule classe par année scolaire. Cela signifie:
- Index unique conditionnel sur (academic_year_id, head_teacher_id) WHERE head_teacher_id IS NOT NULL
- Ou validation custom dans le FormRequest

### Logique Enseignants Disponibles
```php
$usedTeacherIds = Classe::where('academic_year_id', $yearId)
    ->whereNotNull('head_teacher_id')
    ->pluck('head_teacher_id');

$availableTeachers = User::role('Enseignant')
    ->whereNotIn('id', $usedTeacherIds)
    ->get();
```
