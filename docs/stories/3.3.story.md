# Story 3.3: Vue Comparative des Coefficients entre Séries

## Status: Ready for Review

## Story

**As an** Admin,
**I want** voir un tableau comparatif des coefficients entre les différentes séries pour un même niveau,
**so that** je puisse vérifier la cohérence de la configuration et voir rapidement les différences.

## Acceptance Criteria

1. Un écran "Comparaison des Coefficients" affiche un tableau croisé :
   - Lignes : Matières
   - Colonnes : Séries (A, C, D) - pour le niveau sélectionné
   - Cellules : Coefficient (ou "-" si la matière n'est pas enseignée dans cette série)
2. Le tableau est disponible uniquement pour les niveaux 1ère et Tle
3. La somme des coefficients est affichée en bas de chaque colonne
4. Les différences significatives sont mises en évidence
5. Un bouton "Exporter en PDF" permet d'imprimer le tableau comparatif

### Logique de Comparaison
```php
// Récupérer tous les coefficients pour un niveau donné
$coefficients = SubjectClassCoefficient::where('level_id', $levelId)
    ->with(['subject', 'series'])
    ->get()
    ->groupBy('subject_id');

// Structurer en tableau croisé
$activeSeries = Series::active()->get();
$comparison = [];
foreach ($coefficients as $subjectId => $items) {
    $row = ['subject' => $items->first()->subject];
    foreach ($activeSeries as $series) {
        $coeff = $items->firstWhere('series_id', $series->id);
        $row['coefficients'][$series->code] = $coeff?->coefficient ?? null;
    }
    $comparison[] = $row;
}
```

### Restriction Niveaux
Cet endpoint n'est pertinent que pour les niveaux 1ERE et TLE. Si un niveau sans séries est passé (6E-2NDE), retourner une erreur 422.
