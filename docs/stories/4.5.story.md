# Story 4.5: Vue Enseignant - Mes Matières et Classes

## Status: Aproved

## Story

**As an** Enseignant,
**I want** voir la liste des matières et classes qui me sont affectées pour l'année en cours,
**so that** je sache pour quelles classes et matières je dois saisir des notes.

## Acceptance Criteria

1. Un écran "Mes matières et classes" affiche la liste des affectations pour l'année active
2. La liste affiche : Matière (code + nom), Classe (nom complet), Nombre d'élèves, Coefficient, Heures/semaine
3. Un filtre par semestre est disponible
4. Chaque ligne a un lien "Voir les élèves" pour accéder à la liste des élèves de la classe
5. Cette liste sera utilisée par le Module Notes pour afficher les matières/classes où l'enseignant peut saisir des notes
6. La charge horaire totale est affichée en haut de l'écran

### Récupération de l'enseignant connecté
```php
$teacher = auth()->user();
$assignments = TeacherSubjectAssignment::where('teacher_id', $teacher->id)
    ->whereHas('academicYear', fn($q) => $q->where('is_active', true))
    ->with(['subject', 'class.level', 'class.series'])
    ->get();
```

### Charge Horaire Totale
```php
$totalHours = $assignments->sum(function ($assignment) {
    return SubjectClassCoefficient::where('subject_id', $assignment->subject_id)
        ->where('level_id', $assignment->class->level_id)
        ->where('series_id', $assignment->class->series_id)
        ->value('hours_per_week') ?? 0;
});
```
