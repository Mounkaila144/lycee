# Story 5.2: Configuration du Tableau d'Honneur et Distinctions

## Status: Aproved

## Story

**As an** Admin,
**I want** configurer les seuils pour le tableau d'honneur, les encouragements et les félicitations,
**so that** les distinctions soient correctement attribuées sur les bulletins.

## Acceptance Criteria

1. Dans l'écran "Configuration des Barèmes", une section "Distinctions" permet de configurer : Tableau d'honneur (14/20), Encouragements (12/20), Félicitations (16/20)
2. Le système valide la cohérence : Encouragements <= Tableau d'honneur <= Félicitations
3. Je peux activer/désactiver chaque distinction individuellement
4. Les distinctions activées seront affichées sur les bulletins des élèves correspondants

### Approche Activation/Désactivation
Plutôt que d'ajouter des colonnes boolean séparées, utiliser null comme indicateur de désactivation:
- Si `honor_roll_threshold = null` → distinction désactivée
- Si `honor_roll_threshold = 14.00` → distinction activée

Cela évite d'ajouter 3 colonnes supplémentaires et simplifie la logique.
