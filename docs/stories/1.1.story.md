# Story 1.1: Création et Gestion des Années Scolaires

## Status: Ready for Review

## Story

**As an** Admin,
**I want** créer et gérer les années scolaires de mon établissement,
**so that** je puisse organiser le calendrier scolaire et structurer les inscriptions par année.

## Acceptance Criteria

1. Un écran "Gestion des Années Scolaires" affiche la liste des années scolaires existantes
2. Je peux créer une année scolaire avec : Nom (ex: 2025-2026), Date de début, Date de fin
3. Le nom doit être unique au niveau du tenant (validation backend + frontend)
4. La date de fin doit être postérieure à la date de début
5. Lors de la création, 2 semestres sont automatiquement créés :
   - S1 : début = début année, fin = date configurable (par défaut milieu de l'année)
   - S2 : début = fin S1 + 1 jour, fin = fin année
6. Je peux modifier une année scolaire existante (nom, dates) et ajuster les dates des semestres
7. Je peux supprimer une année scolaire uniquement si aucune classe, inscription ou note n'y est rattachée
8. La liste des années est triable par : Nom, Date de début, Statut
9. Un badge visuel distingue l'année active (badge vert "Active")

### Coding Standards [Source: docs/architecture/coding-standards.md]
- Tous les models: `protected $connection = 'tenant';` + `use SoftDeletes;`

- Form Requests obligatoires (pas de validation inline)
- API Resources obligatoires (jamais retourner models bruts)
- Réponses JSON: `{ message: "...", data: {...} }` pour create/update, collection paginée pour index
- Eager loading pour éviter N+1

### Logique Métier: Auto-création Semestres
Lors du `store()`, après création de l'année scolaire:
```php
// Calcul automatique des dates de semestres
$midDate = $academicYear->start_date->copy()->addMonths(5); // ~fin février
$academicYear->semesters()->createMany([
    ['name' => 'S1', 'start_date' => $academicYear->start_date, 'end_date' => $semester1EndDate ?? $midDate],
    ['name' => 'S2', 'start_date' => ($semester1EndDate ?? $midDate)->addDay(), 'end_date' => $academicYear->end_date],
]);
```
