# Story 6.1: Vue Arborescente de la Structure Académique

## Status: Aproved

## Story

**As an** Admin,
**I want** visualiser la structure académique sous forme d'arbre hiérarchique,
**so that** j'aie une vue d'ensemble de mon établissement.

## Acceptance Criteria

1. Un écran "Vue d'ensemble" affiche la structure académique sous forme d'arbre interactif :
   - Cycle > Niveau > Classes avec détails (PP, effectif/capacité, nb matières)
2. Chaque noeud de l'arbre est cliquable pour afficher les détails
3. Un bouton "Modifier" est disponible sur chaque noeud pour éditer l'entité
4. Les noeuds sont expansibles/collapsibles
5. L'arbre affiche le nombre d'entités enfants et les effectifs entre crochets
6. Les indicateurs visuels montrent l'état (vert = OK, orange = attention, rouge = problème)
7. L'arbre est filtré par année scolaire (défaut : année active)

### Performance
Cet endpoint agrège beaucoup de données. Utiliser eager loading et une seule requête:
```php
$cycles = Cycle::active()
    ->with(['levels.classes' => fn($q) => $q
        ->where('academic_year_id', $yearId)
        ->withCount('enrollments')
        ->with(['headTeacher', 'level', 'series'])
    ])
    ->orderBy('display_order')
    ->get();
```

### Calcul subjects_count par classe
Compter les coefficients configurés pour le level_id/series_id de la classe:
```php
$subjectsCount = SubjectClassCoefficient::where('level_id', $class->level_id)
    ->where('series_id', $class->series_id)
    ->count();
```
